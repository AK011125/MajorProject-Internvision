# Quick Notes Pro 📝

A modern Notes Application built using Next.js, React, TypeScript, and Tailwind CSS.

## 🚀 Live Demo

Add your Vercel link here

## 📌 Features

* Add Notes
* Edit Notes
* Delete Notes
* Search Notes
* Responsive UI
* Data saved in Local Storage
* Fast and clean design

## 🛠️ Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* Local Storage

## 📂 Project Structure

* `app/page.tsx` → Main UI
* `app/layout.tsx` → Layout
* `app/globals.css` → Styles

## ▶️ Run Locally

```bash
npm install
npm run dev
```
